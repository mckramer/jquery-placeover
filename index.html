jQuery.placeover v0.1.0 ALPHA

Text overlay for input fields. A new take on the placeholder shim for jQuery to provide HTML5 behavior. Inspired by OverText for MooTools.

Form

Matches HTML5 placeholder behavior.
Even works on password fields with no extra configuration.
Need a text area?
With default value.

Usage

Quickstart

Include jQuery and the placeover plugin.

Call the placeover method on a selector. Ensure that is called after html is complete (through on ready event or at bottom of page).

Placeover is by default a shim, so it will not introduce this behavior on browsers that natively support the placeholder attribute. However, you can override these settings to ignore supported handling if desired.

<input type="text" placeholder="example placeholder" />
<script src="./jquery.js"></script>
<script src="./jquery.placeover.js"></script>
$('#input').placeover();

Options

Upon initialization, you can pass a set of options through to the placeover method.

$('#input').placeover(options);
Name Type Default Description
class string 'input-placeover' The class to be added to the placeover element
eventNamespace string '.placeover' The namespace to be used when triggering the show/hide events.
hideClass string 'hide' The class to be added to the placeover element when it should not be visible. If set to a falsey value, then will simply toggle display (from none to inline) instead of adding a class.

Methods

Not everything has to be automatic. You can manually trigger the placeover when needed.

$('#input').placeover('show');
Method Description
hide Hides the placeover text.
reposition Resets the position of the placeover text. Useful when manipulating styles of the input box or moving input fields around.
show Shows the placeover text.
update Change the placeover text.

Events

Bind to events to further customize the behavior.

$('#input').on('show.placeover', showHandler);
Event Description
show This event fires immediately when the show instance method is called. You can prevent the method from being shown by preventing default on the event.
shown This event is fired when the placeover element has been made visible to the user.
hide This event fires immediately when the hide instance method is called. You can prevent the method from being hidden by preventing default on the event.
hidden This event is fired when the placeover element has been hidden from the user.
